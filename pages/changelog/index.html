2026-02-02

Cheep Cheep/Discord

- Match announcements are now posted to the schedule channel
- Cheep Cheep will now post new news articles from umkl.co.uk to the news-articles channel
- SP now has a leaderboard in /leaderboards
- New command - /leave-match which removes yourself from a match you signed up for
- Changed the dimensions of the graphic used as the image for Discord events
- The new SP formula rewards more SP to all players, with a smaller variation of points for players on the winning team based on individual scores, compared to players on the losing team that now receive SP that scales more with individual score (this new formula applies to matches from 01/12/2025)
- (Board) Cheep Cheep now auto organises the team order based on the current standing when using /create-match
- (Board) /add-match-results now ends the match and discord event when results are verified
- (Board) New command - /generate-yt-thumbnails which generates the YouTube thumbnails for match livestreams
- (Board) Fixed a bug where test matches couldn't be created for teams not participating in the current season
- (Board) Add commands /add-next-round-pairings and /start-next-round
- Various bug fixes

UMKL website

- Improved liveresults
- The 6v6 Score Calculator https://umkl.co.uk/pages/tools/6v6scorecalculator/ can now generate graphs for race scores over a match, these graphs and links can be shared
- Some new pages and features on the website
- Added keyboard shortcuts to more pages on umkl.co.uk
- You can now click match results on the matches page on umkl.co.uk, and share the results graph
- Add a combo randomiser tool
- Merge FAQ with Rules to make the Rules/Guides page
- Added a page that shows changelogs on the website
- Added more easter eggs
