Cheep Cheep Updates

17/07/2025 | Discord Test Matches
Written by zy

Hi all! Here are a few updates.

It's been a few months since the announcement of @Cheep Cheep app, since then a lot of new commands and features have been added.

Work is being done constantly to improve the stability of our systems we use for UMKL, so we hope everyone is happy with the continued development!

The most important change from the previous message is that the /register has been removed, as it is no longer required to use /join when you're joining a team. A full changelog will be listed at the end of this page.

A few test matches have occurred recently and they've gone smoothly, so more test matches will follow! So be sure to check the matches page for upcoming matches!

Below are some newly added commands that can be used in the server!

Changelog for the 17/07/2025 update
  • /register removed, it is no longer required to use /join
  • New command - /leaderboards
  • Formatted YYYY-MM-DD to DD/MM/YYYY in embeds (only for outputted text, not for command inputs)
  • Added the match versus image on the sign-up embed
  • Changed the date in the sign-up embed to a timestamp
  • New command - /change-team-colour (board only)
  • Internal changes to SQL/DB threading
  • Internal changes to the API used on umkl.co.uk
  • Tweaked graphics to add shadowing beneath text to improve contrast and readability
  • New command - /generate-stream-overlay
  • Fixed /team-profile
  • New commands (used for streaming overlay):
    • /add-results-to-liveresults
    • /edit-results-in-liveresults
    • /reset-liveresults
  • New command - /submit-room-code
  • New command - /report-disconnection
  • Added some easter eggs

This article is a summary of an announcement made on the UMKL Discord server, you can join the server here.
