Discord/Cheep Cheep Commands Documentation

Documentation for all Cheep Cheep commands on the Discord server

Commands Usable by All Members

All players can use these commands

General

/join

Joins a team, required to participate in UMKL matches.

/user-profile

View a specific user's profile (yourself or someone else's), showing their team, wins, matches played, podiums, highest finish, career points and SP.

/team-profile

View a team's profile and stats, including location, institution, season position, win/loss record, points, penalties and championships.

/leaderboards

View different UMKL leaderboards: Season/Lifetime Points and Season/Lifetime Best Match Score (both Team/Individual), and SP.

/view-sp-from-match

View how much SP was earned in a specific match you played.

Note
SP is earned by playing in matches and being active in the UMKL Discord server.
/export-my-data

Download all data associated with your account as a zip file.

Match

Note
Signing up for a match isn't a command - use the "Sign Up" button in your team's announcements channel, then fill in your display name and friend code to be added to the match roster.
/submit-room-code

Share the room code for the match with both teams, used when a room is opened before the match starts so everyone can join.

/report-disconnection

Report a disconnection for yourself or someone else on your team.

/leave-match

Remove yourself from a match you've signed up for.

/generate-stream-overlay

Generate a stream overlay for yourself or someone else, designed for streaming UMKL matches and for use with livestreaming from OBS.

Live Score

Commands for use with the live score graphic, as shown on livestreams

/add-live-score-positions

Add the results of a race, which will update the live score graphic.

/edit-live-score-positions

Edit the results of a previous race, which will update the live score graphic.

/clear-live-score

Reset the live score streaming graphic, setting the score to 0-0.

Commands Only Usable by Board Members

Only board members can use these commands

Teams

/create-team

Create a new team to represent a university. This automatically creates the team's Discord channels, roles and category.

/change-team-colour

Change the colour a team uses.

/change-team-name

Change the name of a team.

/enter-team-into-season

Enter a team into the current season.

/remove-team-from-season

Remove a team from the current season.

/request-stream-key

Get your team's stream key, used for livestreaming matches.

Match

/create-match

Create a new match between two teams. This generates a versus image, creates a Discord scheduled event, and posts sign-up embeds in each team's channel.

/edit-match-title

Edit an existing match's title.

/edit-match-description

Edit an existing match's description.

/edit-match-date

Reschedule an existing match to a new date.

/edit-match-time

Edit an existing match's time.

/add-yt-link

Add a YouTube link to a match's livestream.

/remove-yt-link

Remove a YouTube link from a match's livestream.

/substitute-player

Substitute one player in a match for another, before the match starts.

/remove-player

Remove a player from a match.

/add-match-results

Add the results of your team's players in a match. Results go through an Approve/Reject/Add Penalties verification step in the board's results channel before being finalised.

/generate-yt-thumbnails

Generate the YouTube thumbnails for both teams in a match's livestream.

Season

/start-season

Start a new season of the UMKL.

/start-next-round

Start the next round of the season and send out match organisation threads.

/add-next-round-pairings

Add the team pairings for the next round.

Admin

/restart

Restart the Cheep Cheep app.

/request-board-password

Get the board password.

/request-media-password

Get the media team password.
