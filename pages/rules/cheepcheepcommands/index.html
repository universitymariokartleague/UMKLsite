Discord/Cheep Cheep Commands Documentation

Documentation for all Cheep Cheep commands on the Discord server, split into sections for players and for board members

Commands usable by all members

All players can use these commands


General

/join
Joins a team, this a required to participate in UMKL matches.

/user-profile
View a specific user's profile (yourself or someone elses)

/team-profile
View a team's profile and stats

/leaderboards
View different UMKL leaderboards, such as Season Points and Lifetime Points (Team/Individual) and also SP


Match

/submit-room-code
Share the room code for the match to both teams, used when a room is opened before the match starts so everyone can join

/report-disconnection
Use this command to report a disconnection for yourself or someone else on your team

/generate-stream-overlay
Generate a stream overlay for yourself or someone else, designed for streaming UMKL matches and for use with livestreaming from OBS

Live Score

Commands for use with the live score graphic, as shown on livestreams


/add-live-score-positions
Add the results of a race, which will update the live score graphic

/edit-live-score-positions
Edit the results of a previous race, which will update the live score graphic

/clear-live-score
Reset the live score streaming graphic, setting the score to 0-0
