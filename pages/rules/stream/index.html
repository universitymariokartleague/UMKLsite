Streaming Guidelines

Guidelines for streamers and instructions on how to set up OBS

OBS Set Up

How to set up OBS with the UMKL overlay and stream to YouTube


Overview

Use /generate-stream-overlay on the UMKL Discord server to produce a stream overlay for yourself or somebody else, this overlay will be for UMKL streams.

Adding Sources

In OBS, adding sources is how you will construct the visuals on your stream.
List of sources

  • Discord VC Overlay (follow the instructions on https://streamkit.discord.com/overlay)
  • Score Overlay (https://live.umkl.co.uk/ or https://liverev.umkl.co.uk/ depending on the team order)
  • Stream Overlay (Can be generated with /generate-stream-overlay in the UMKL Discord server)
  • Video Capture Device (Capture Card)
Sources at the top will display on top of sources below it.
A Note About Audio
There are two ways to hear audio from Discord in your stream.
You can use Desktop Audio to capture audio from Discord, but it is recommended to use add an Application Audio Capture (BETA) source and directly capture audio from the Discord application, as this will avoid any potential double audio issues.
Be sure to mute any audio sources that aren't in use.

If you want to hear game audio without the need of splitters, you can right-click
Video Capture Device in the Audio Mixer panel, select Advanced Audio Properties and set Audio Monitoring for the capture card to Monitor and Output to hear game audio through OBS and your computer.

Entering a Stream Key

To enter a stream key in OBS, navigate to Settings and click the Stream tab. Select YouTube - RTMPS in the service dropdown, click Use Stream Key (advanced) and paste in your team's stream key.

Note
Please contact a board member to recieve your team's stream key if you don't have it yet.

Multiple Profiles

In OBS you can create multiple profiles. You can change them with the 'Profile' dropdown menu. This is useful if you need to switch streaming destinations (ie: personal account and UMKL channel).

Notes

  • Please note that the stream overlay (and the title of the stream) will contain your Discord username, to signify which team's perspective the stream is from, this is so it matches the username shown on results graphics and such. Please ensure your DM settings on Discord are as you'd like bearing this information in mind
  • Ideally we would want at least one person playing on each team to be able to stream the match (this isn't a requirement but it's ideal)
  • Ensure your bitrate is set high enough for the stream to look good. Bitrate Guidelines on YouTube Help
    At 1080p60 (H.264), 12 Mbps bitrate is recommended. If your connection allows it, it can be benificial to stream at 1440p or 4K resolutions for access to higher bitrates on YouTube.
  • Do not show the room code on-screen, block it from the screen if needed
